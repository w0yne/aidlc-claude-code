# AI-DLC Workflow Command

Trigger the full AI-DLC (AI-Driven Development Life Cycle) methodology for the following task:

$ARGUMENTS

## What This Command Does

This command activates the complete AI-DLC adaptive software development workflow as defined in `CLAUDE.md`. The methodology will:

1. **Display the AI-DLC welcome message** from `aidlc-workflows/common/welcome-message.md`
2. **Load common workflow rules** (process overview, content validation, question format guide, session continuity)
3. **Execute the INCEPTION PHASE** - analyzing your workspace, gathering requirements, and planning the approach
4. **Execute the CONSTRUCTION PHASE** - designing, implementing, and testing your solution
5. **Maintain a complete audit trail** of all decisions and interactions

## Full Methodology

The complete workflow instructions are in `CLAUDE.md`. Key workflow files are in `aidlc-workflows/`:

- `aidlc-workflows/common/` - Cross-phase guidelines (content validation, question format, error handling, etc.)
- `aidlc-workflows/inception/` - INCEPTION phase workflow files
- `aidlc-workflows/construction/` - CONSTRUCTION phase workflow files
- `aidlc-workflows/operations/` - OPERATIONS phase workflow files (placeholder)

## Starting the Workflow

Follow all instructions in `CLAUDE.md` to execute the AI-DLC methodology for the task described above. Begin with the MANDATORY steps: display the welcome message, load common rules, then start Workspace Detection.
